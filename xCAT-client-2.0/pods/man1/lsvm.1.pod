=head1 NAME
 
B<lsvm> - Lists partition profile information for HMC- and IVM-managed nodes.
 
=head1 SYNOPSIS
 
I<lsvm [-h| --help]>

I<lsvm  [-v| --version]>


I<lsvm [-V| --verbose] noderange>


=head1 DESCRIPTION
 The lsvm command lists all partition profiles defined for the partitions specified in noderange. If noderange is an FSP, all the partitions associated with that FSP are displayed.

=head1 OPTIONS


B<-h>          Display usage message.

B<-v>          Command Version. 

B<-V>          Verbose output. 


=head1 RETURN VALUE

0 The command completed successfully.

1 An error has occurred.
