=head1 NAME

rpower - remote power control

=head1 SYNOPSIS

B<rpower>
I<noderange>
B<on>|B<off>|B<stat>|B<state>|B<reset>|B<boot>|B<cycle>

B<rpower>
[B<-h>|B<--help>]

B<rpower>
[B<-v>|B<--version>]

=head1 DESCRIPTION

I<rpower> controls the power for a single or range of nodes.

=head1 OPTIONS

=over 4

=item B<on>

Turn power on.

=item B<off>

Turn power off.

=item B<stat>|B<state>

Print the current power state/status.

=item B<reset>

Send a hard reset.

=item B<boot>

If off, then power on.
If on, then hard reset.
The option is recommended over B<cycle>.

=item B<cycle>

Power off, then on.

=item B<-h>, B<--help>

Help.
Prints out a brief help message.

=item B<-v>, B<--version>

Version.
Display the version number.

=back

=head1 ENVIRONMENT

=over 4


=back
